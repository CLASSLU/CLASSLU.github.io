# 路漫漫科技主页（单页）

这是一个静态单页个人公司主页，定位为：

- 个人技术公司展示
- 工程能力矩阵呈现
- 当前探索方向与代表项目说明
- 仅中文内容

## 文件说明

- `index.html`：页面结构与文案
- `styles.css`：视觉样式、响应式与动效
- `script.js`：移动导航、滚动状态与入场动效

## 本地预览

方式 1：直接打开 `index.html`。  
方式 2：使用静态服务：

```bash
npx --yes serve . -l 4173
```

然后访问 `http://localhost:4173`。

## 内容维护

- 更新文案：编辑 `index.html`
- 更新视觉主题：编辑 `styles.css` 中 `:root` 变量
- 更新联系邮箱：编辑 `index.html` 中 `mailto` 链接
