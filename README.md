# 武汉久玩电商官网（单页）

这是一个静态单页官网，包含：

- 中英双语切换（默认中文，自动记忆语言）
- 响应式布局（桌面与移动端）
- 品牌背书导向的信息结构
- 联系方式展示（邮箱与地址）

## 文件说明

- `index.html`：页面结构与语义化内容
- `styles.css`：视觉样式、响应式与动画
- `script.js`：双语切换、移动导航、滚动动效

## 本地预览

方式 1：直接双击 `index.html` 在浏览器打开。  
方式 2：使用静态服务：

```bash
npx --yes serve . -l 4173
```

然后访问 `http://localhost:4173`。

## 内容维护

- 修改文案：编辑 `script.js` 的 `translations` 对象
- 修改色彩与字体：编辑 `styles.css` 的 `:root` 变量
- 修改模块结构：编辑 `index.html`
